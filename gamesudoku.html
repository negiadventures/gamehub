How to Play

  • Fill the grid so that each row, column, and 3x3 box contains digits 1-9
  • Click a cell to select it, then use the numpad or keyboard to enter a number
  • Enable Notes to add pencil marks to cells
  • You have 3 mistakes before the game ends
  • Red cells indicate conflicts with row, column, or box
