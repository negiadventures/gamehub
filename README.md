# GameHub

GameHub is a collection of browser-based HTML5 games in a single static site.

## Included games

- 2048
- Breakout
- Connect 4
- Copter
- Flappy Bird
- Minesweeper
- Memory
- Pac-Man
- Pong
- Snake
- Space Invaders
- Sudoku
- Tetris
- Tic-Tac-Toe

## Run locally

This project is static HTML/CSS/JS, so no build step is required.

You can open `/home/runner/work/gamehub/gamehub/index.html` directly in a browser, or serve the folder with any static server.

## Project structure

- `index.html` — main game hub page
- `*.html` — individual game pages
- `styles.css` — shared styles
- `leaderboard.js` / `multiplayer.js` — shared gameplay scripts
